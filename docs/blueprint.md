# **App Name**: Useless Facts

## Core Features:

- Fact Generation: Generate a completely useless and random fact.
- Fact Display: Display the generated fact in a readable format.
- Share Fact: Option to easily share the useless fact on social media.

## Style Guidelines:

- Primary color: A desaturated green (#8FBC8F) to give a quirky and unusual feel.
- Background color: Dark grayish-green (#2E492E), a dark scheme providing contrast.
- Accent color: A muted olive green (#A8B878), a low contrast colour from an adjacent hue for gentle highlighting.
- Body and headline font: 'Inter', a sans-serif for readability and a modern feel.
- Use simple, outline-style icons for share options.
- Simple and clean layout to focus on the useless fact, with sharing options below.
